# GT-PHYS-4604

PHYS 4604 — Professional Development

This repository contains course materials and web pages for PHYS 4604 (Professional Development), a 1-credit, in-person course offered at the School of Physics, Georgia Institute of Technology. The files here provide the public-facing syllabus, welcome page, assignment examples, and supplemental resources intended for students and instructors.

Contents
- `welcome-page.html` -- short home/welcome page for the course.
- `syllabus.html` -- full course syllabus and policy information.
- `assignments/` -- HTML pages for individual assignments (resume, online profile, research proposal, personal statement, external participation).
- `course-files/phys-4604-syllabus-fall26.tex` -- LaTeX source for the syllabus.
- `supplemental/` -- additional resources (resume/CV template, technical-writing guidance).
